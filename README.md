# Tic Tac Toe

Play the famous Tic Tac Toe game with CLI.
This is a project from [The Hacking Project](https://www.thehackingproject.org) courses.

## How to play

```bash
$ ruby tictactoe.rb
```

## Requirements

- `ruby '2.7.7'` and above
